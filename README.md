# SMASH | Premium Badminton E-commerce

A professional, high-end e-commerce platform dedicated to badminton gear.

## Features

- **Premium UI**: Minimalist, high-contrast design using Public Sans and Outfit fonts.
- **Dynamic Shop**: Category filtering and live product stock.
- **Shopping Bag**: Complete cart management with live total calculation.
- **Secure Auth**: JWT-based login and signup.
- **Full-Stack**: React frontend with Node.js/Express and MongoDB/Prisma backend.

## Tech Stack

- **Frontend**: React (Vite), Plain CSS
- **Backend**: Node.js, Express, Prisma ORM
- **Database**: MongoDB

## Getting Started

### Backend

1. `cd backend`
2. `npm install`
3. Configure `.env` (MongoDB URI)
4. `node seed-native.js` (Populate initial data)
5. `npm run dev`

### Frontend

1. `cd frontend`
2. `npm install`
3. `npm run dev`
